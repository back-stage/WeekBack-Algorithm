# WeekBack-Algorithm
WeekBack에서 진행하는 알고리즘 공동 목표 뽀개기

# 기수
## 1기
* WeekBack 기수에 맞춰 진행, WeekBack 모든 멤버 참여
* 2021.02.01 ~ 2021.02.21 (5 ~ 7주차 진행)

## 2기
* WeekBack 기수에 맞춰 진행
* 2021.03.08 ~ 2021.04.25 (1 ~ 8주차 진행)

## 3기
* WeekBack 중 희망 멤버만 참여
* 2021.05.03 ~ 2021.07.11

## 4기
* WeekBack 모든 멤버 참여
* 2021.07.19 ~ 2021.09.19
